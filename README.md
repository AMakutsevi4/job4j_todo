# job4j_todo

Учебный проект TO DO — это простой список дел или список задач. Запиши в него все свои важные дела, чтобы не забыть.

#### Технологии проекта ####
Spring Boot,
PostgreSQL,
Hibernate,
Java-16,
Maven,
Checkstyle.
